# Campus-Event-Hub
Frontend: - React.js (Vite) - Tailwind CSS - React Router - Axios  Backend: - Node.js - Express.js - MongoDB (Mongoose) - JWT Authentication  Other Features: - QR code generation (qrcode npm package) - QR code scanning (html5-qrcode library) - Role-based access (Student / Organizer) - CSV export for registrations - Clean modern UI
